{%- comment -%} Every Jekyll page declares `title:` and `description:` in its own frontmatter. og:type is derived from layout (post -> "article", else "website"). Page titles in frontmatter omit the brand suffix; we append " - Codatus" here unless the title already contains "Codatus" (home page, scorecards). Same code path for every page. {%- endcomment -%} {%- if page.layout == "post" -%}{%- assign og_type = "article" -%}{%- else -%}{%- assign og_type = "website" -%}{%- endif -%} {%- assign full_title = page.title -%} {%- unless page.title contains "Codatus" -%} {%- assign full_title = page.title | append: " - Codatus" -%} {%- endunless -%} {{ full_title }}
