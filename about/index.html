About Codatus

Codatus is a GitHub repo scorecard for platform teams. The scanner reads repository metadata across every repo in an organization and produces one page showing how each repo performs on branch protection, code ownership, CI, and documentation.

Why this exists

Codatus was built by Peter Novak, an independent developer based in Slovakia, Europe, to solve a problem he kept hitting at previous orgs: answering "are our repos in a reasonable state?" without spending a week on spreadsheets. Enterprise tools answer it through a full developer portal (Backstage, Cortex, OpsLevel). Smaller platform teams either skip the question or run a one-off audit script that goes stale within a quarter.

Codatus is the option in between. One install, one scorecard, refreshable on demand.

The scanner is open source on GitHub. For questions, write to hello@codatus.com.
