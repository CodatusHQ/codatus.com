About Codatus

Codatus is an alert-first monitor for GitHub. A read-only GitHub App watches the controls on your organization, visibility, branch protection, required checks, reviewers, bypass actors, and secret scanning, and sends a routed alert the moment one of them changes.

Why this exists

GitHub records every one of these changes in its audit log and notifies no one. A repo can go public, branch protection can come off, a required check can be dropped, all silently. The platforms that catch this assume a security team to run them and a budget to match; free tools catch only pieces.

Codatus was built by Peter Novak, an independent developer based in Slovakia, Europe, for the team in between: too big to leave its repos unwatched, too small for enterprise security tooling.
